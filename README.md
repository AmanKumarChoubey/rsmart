RS Mart ERP is a full-stack enterprise-level store management system designed to handle end-to-end retail operations. The platform supports multiple roles including Admin, Manager, Shopkeeper, and Customer, each with a dedicated dashboard and role-based access.

Key features :
           Include store and employee management, product and inventory control, order handling, role-based authentication, and customer shopping experience inspired by Amazon-style UI/UX. The system is built with a scalable backend and a modern, responsive frontend to ensure performance, security, and usability.
           
Tech Stack: 
            Java, Spring Boot, Spring Security, JPA/Hibernate, Thymeleaf, HTML, CSS, JavaScript, SQL etc.
